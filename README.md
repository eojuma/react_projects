# My React Projects ⚛️

A simple and clean collection of React applications. Each folder in this repository is a standalone project.

## 🛠️ Tech Stack
* **Framework:** React.js (Vite)
* **Styling:** CSS / Tailwind CSS
* **Language:** JavaScript

## 🏃 Getting Started

### 1. Clone the Repository
git clone [https://github.com/eojuma/react_projects.git](https://github.com/eojuma/react_projects.git)
### 2.Navigate to the repository/folder name
cd react_projects
### 3. Install dependencies
npm install
### 4.Launch the development server
npm run dev
### 5.Build for production 
npm run build
